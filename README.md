1. 增加海报墙标题
2. 增加豆瓣评分（需要搭配魔改版 tmdb 助手） https://github.com/wabisabi926/plugin.video.themoviedb.helper/tree/douban
3. 更新中文翻译


## 安装

通过 Martian 插件仓库安装，以启用自动更新。

Kodi 文件管理器源：
https://martian89.github.io/repository.martian/

直接 ZIP 安装：
https://martian89.github.io/repository.martian/repository.martian-1.0.0.zip

安装说明：

1. 在 Kodi 设置 > 系统 > 插件中启用 "未知来源"
2. 使用上面链接的 zip 文件或文件管理器源安装仓库
3. 从该仓库安装最新版本的 Arctic: Zephyr (martian)
